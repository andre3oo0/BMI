BMI Calculator
This is used to calculate and classify the Body Mass Index (BMI) of a large number of people.
 The program prompts the user to enter the weight and height of each person, calculates the BMI, and  displays the BMI  with the appropriate weight category (underweight, normal weight, overweight, or obese).

 Step 1: Usage Clone the repository or download it to your local computer.
 Step 2: Open the project in your preferred Java development environment preferably NetBeans
 Step 3: Run the IDProgram.java file to start the program.
 Step 4: Follow the on-screen instructions to enter your personal information, such as your name, phone number, email address, and ID number.
